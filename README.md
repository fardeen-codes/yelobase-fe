# Yelobase

A production-ready Next.js 15 company website scaffold with App Router, TypeScript, Tailwind CSS, Shadcn-style components, Framer Motion, React Hook Form, Zod, and WordPress REST API integration.

## Getting Started

1. Install dependencies
   `npm install`
2. Copy `.env.local.example` to `.env.local` and fill in your WordPress endpoint and revalidation secret.
3. Run the dev server
   `npm run dev`
4. Build the app
   `npm run build`
